# Техническое задание

## 1. Цель проекта

Провести тестирование веб-приложения Wikipedia и выявить возможные ошибки в работе интерфейса и функциональности.

## 2. Задачи проекта

- изучить функциональность приложения
- составить чек-лист тестирования
- разработать тест-кейсы
- провести тестирование
- оформить отчет

## 3. Целевая аудитория

- студенты
- преподаватели
- пользователи интернета, ищущие информацию

## 4. Проблемы, которые решает приложение

Wikipedia предоставляет доступ к свободной энциклопедии и помогает пользователям быстро находить информацию.

## 5. Аналоги

- Britannica
- Scholarpedia
- Google Knowledge

## 6. Тип приложения

Веб-приложение, доступное через браузер.
